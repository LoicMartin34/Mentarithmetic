# MentaRithmetic
Website about mental arithmetic training.

## 4 difficulties : 
  - easy : addition and subtraction
  - medium : easy + multiplication
  - hard : medium + division
  - expert : hard + modulo
  
The modulo is the rest of an euclidian division.
Ex: 10 modulo 3  or 10 % 3  = 1
